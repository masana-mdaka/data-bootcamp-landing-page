8 Week Data Bootcamp

For students & professionals (with no prior experience) looking to pivot to data engineering / data analysis / data science

What We'll Cover:

  • Programming in Python
  • Programming in SQL
  • Introduction to Cloud
  • Preparation for Cloud (AWS) certification
  • Data visualisation using Power BI
  • Development of a real-world project and portfolio on GitHub
  • LinkedIn, CV and Cover Letter Optimisation for job readiness

Detailed Weekly Plan

Week 1 – Python, SQL, Git: Data Foundations

  • Tools: VS Code, Jupyter, GitHub, Terminal
  • Topics: Python basics, SQL fundamentals, Git/GitHub workflows
  • Mini Project: Python + SQL report

Week 2 – Data Engineering and Analysis

  • Libraries: Pandas, NumPy
  • Topics: ETL vs ELT, data wrangling, statistics
  • Mini Project: EDA notebook

Week 3 – Visualization & Dashboarding

  • Tools: Matplotlib, Seaborn, Plotly, Power BI
  • Topics: Charting best practices, storytelling, dashboards
  • Mini Project: Dashboard from EDA

Week 4 – Databases & Cloud Intro

  • Tools: MySQL, AWS basics
  • Topics: SQL vs NoSQL, cloud storage, pipeline deployment

Week 5 – Intro to Data Science

  • Libraries: scikit-learn, statsmodels
  • Topics: Regression, classification, model evaluation
  • Mini Project: Build a machine learning model

Week 6 – Capstone Project

  • Goal: Complete data project with cloud + ML or analytics
  • Deliverables: GitHub repo, README, visualizations

Weeks 7-8 – Career Prep & Certification

  • Cert Prep: AWS Cloud Practitioner / Data Engineer Associate
  • Career Tools: CV, LinkedIn, mock interviews, GitHub polish

Tools & Accounts Needed

  • Python
  • Jupyter or VS Code
  • GitHub account
  • AWS Cloud account
  • MySQL
  • Power BI

Outcomes by End of Bootcamp

  • 1–2 Projects on GitHub (ETL + ML/EDA/Visualization)
  • Solid grasp of Python, SQL, data wrangling
  • Intro to ML and model evaluation
  • Familiarity with cloud services + certification prep
  • Portfolio: GitHub + LinkedIn optimized
  • CV aligned for junior data/engineering roles
